📈 Debug Results & Root Cause Analysis

✅ Pattern Found

All successful tests (4,5,6,8) use:
• Electron 28.0.0+ OR
• npm ci (instead of npm install)

❌ Root Cause Identified

Electron 27.0.0 + npm install = 100% failure rate
This combination fails on macOS 14 due to native module build permissions.

⚠️ Recommendations

1. Always use "npm ci" instead of "npm install" on macOS 14+
2. Update Electron baseline to 28.0.0
3. Skip boot video optional for M1 optimization
